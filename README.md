## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone and setup**
   ```bash
   cd old-portfolio
   npm install
   ```

2. **Development**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000)

3. **Build for production**
   ```bash
   npm run build
   npm start
   ```

### Deployment to Vercel

1. **Connect to Vercel**
   ```bash
   npx vercel
   ```

2. **Configure environment** (if needed)
   - Add environment variables in Vercel dashboard
   - Update `vercel.json` for custom configuration

3. **Deploy**
   ```bash
   npx vercel --prod
   ```
